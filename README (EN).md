# Minimal Spotify Visualizer

A clean and minimal Spotify visualizer for Rainmeter.

## Installation

1. Install **Rainmeter**  
   [https://www.rainmeter.net/](https://www.rainmeter.net/)

2. Download and install the skin from the **Releases** tab

3. Install the **WebNowPlaying** plugin  
   [https://github.com/keifufu/WebNowPlaying-Rainmeter/releases/tag/2.0.7](https://github.com/keifufu/WebNowPlaying-Rainmeter/releases/tag/2.0.7)

4. Install the **WebNowPlaying** browser extension  
   [https://chromewebstore.google.com/detail/webnowplaying/jfakgfcdgpghbbefmdfjkbdlibjgnbli](https://chromewebstore.google.com/detail/webnowplaying/jfakgfcdgpghbbefmdfjkbdlibjgnbli)

5. Load the skin: `MinimalSpotifyVisualizer.ini`

## Features

- Displays track album art with rounded corners
- Shows artist name and track title
- Shows current playback time and duration
- Progress bar
- Real-time audio visualizer
- Playback controls: Play/Pause, Previous, Next

You can control Spotify playback directly from the widget without opening the Spotify app.
