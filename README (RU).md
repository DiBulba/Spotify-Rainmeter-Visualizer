# Minimal Spotify Visualizer

Минималистичный визуализатор для Spotify под Rainmeter.

## Установка

1. Установите приложение **Rainmeter**  
   [https://www.rainmeter.net/](https://www.rainmeter.net/)

2. Установите сам скин из вкладки **Releases**

3. Установите плагин **WebNowPlaying**  
   [https://github.com/keifufu/WebNowPlaying-Rainmeter/releases/tag/2.0.7](https://github.com/keifufu/WebNowPlaying-Rainmeter/releases/tag/2.0.7)

4. Установите расширение **WebNowPlaying** для браузера  
   [https://chromewebstore.google.com/detail/webnowplaying/jfakgfcdgpghbbefmdfjkbdlibjgnbli](https://chromewebstore.google.com/detail/webnowplaying/jfakgfcdgpghbbefmdfjkbdlibjgnbli)

5. Запустите скин: `MinimalSpotifyVisualizer.ini`

## Функционал

- Отображение обложки трека со скруглёнными углами
- Имя исполнителя и название трека
- Текущее время и длительность трека
- Прогресс-бар
- Аудио-визуализатор
- Управление воспроизведением: Play/Pause, Previous, Next

Вы можете управлять треками прямо из виджета, не открывая приложение Spotify.
