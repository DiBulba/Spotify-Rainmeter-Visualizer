# Гайд по настройке Virtual Audio Cable для Minimal Spotify Visualizer

## Обзор

Этот гайд поможет вам настроить Minimal Spotify Visualizer так, чтобы он реагировал **только на музыку из Spotify**, используя VB-Audio Virtual Cable, отделяя системные звуки.

## Что вам понадобится

1. **VB-Audio Virtual Cable** (бесплатно) - [Скачать](https://vb-audio.com/Cable/index.htm)
2. **Rainmeter** - [Скачать](https://www.rainmeter.net/)
3. **EarTrumpet** (рекомендуется, бесплатно) - [Скачать](https://www.microsoft.com/en-us/p/eartrumpet/9nblggh516ft)
4. **Minimal Spotify Visualizer** - Уже установлен с этим скином

## Пошаговая настройка

### Шаг 1: Установка VB-Audio Virtual Cable

1. Скачайте и установите VB-Audio Virtual Cable
2. Перезагрузите ПК (это необходимо)
3. Теперь вы должны увидеть два устройства:
   - **CABLE Input** - Сюда идёт звук ВНУТРЬ (Spotify отправляет сюда)
   - **CABLE Output** - Отсюда звук выходит НАРУЖУ (вы слушаете отсюда)

### Шаг 2: Маршрутизация Spotify на Virtual Cable с помощью EarTrumpet

1. **Найдите иконку микшера звука в трее Windows** (иконка динамика внизу справа)

2. **Нажмите правой кнопкой мыши на иконку динамика** и выберите **"Микшер громкости EarTrumpet"**

3. **Откроется окно микшера громкости EarTrumpet**

4. **Найдите Spotify в списке приложений** и нажмите на него **правой кнопкой мыши**

5. **В контекстном меню выберите опцию для выбора устройства вывода**

7. **Выберите "CABLE Input (VB-Audio Virtual Cable)"** из списка доступных устройств

Теперь звук Spotify идёт только в Virtual Cable, не в ваши основные динамики!

### Шаг 3: Установка выхода Virtual Cable на ваши динамики

1. Нажмите правой кнопкой на значок громкости в трее → **Параметры звука**
2. Прокрутите вниз → **Дополнительные параметры** → **Громкость и качество устройства**
3. Найдите **CABLE Output**
4. Установите его как устройство вывода для ваших динамиков
5. (Опционально) Можете включить "Прослушивать это устройство", если хотите слышать Spotify через микрофон

### Шаг 4: Настройка визуализатора для прослушивания Virtual Cable Input

1. Загрузите или обновите скин **Settings** в Rainmeter
2. Прокрутите вниз до раздела **— Audio Devices**
3. Нажмите кнопку **"Refresh list"** (если устройства не показаны)
4. Вы должны увидеть что-то вроде:
   - CABLE Input (VB-Audio Virtual Cable)
   - CABLE Output (VB-Audio Virtual Cable)
   - Realtek Audio (или ваша основная звуковая карта)
5. **Нажмите на "CABLE Input"** - она должна стать зелёной

### Шаг 5: Проверка работы

1. Включите музыку в Spotify
2. Визуализатор должен реагировать **только на музыку Spotify**
3. Откройте YouTube или игру
4. Визуализатор должен **НЕ реагировать** на эти звуки ✅

Если не работает:
- Убедитесь, что CABLE Input выбран (зелёная кнопка)
- Убедитесь, что Spotify выводит звук в CABLE Input через EarTrumpet
- Обновите скин визуализатора (F5 или правый клик → Обновить)

## Решение проблем

### "Устройство не найдено"
- Нажмите кнопку **"Refresh list"** в Settings
- Убедитесь, что VB-Cable установлен и работает

### Визуализатор всё ещё реагирует на все звуки
- Проверьте, что выход Spotify установлен на CABLE Input через EarTrumpet
- Убедитесь, что вы выбрали правильный кабель в Settings
- Попробуйте нажать "Use default", а потом снова выбрать CABLE Input

### Звук Spotify не слышно
- Проверьте, что CABLE Output установлен как устройство воспроизведения
- Или включите "Прослушивать это устройство" на CABLE Output, чтобы слышать через микрофон

### EarTrumpet не показывает Spotify
- Убедитесь, что Spotify реально проигрывает музыку
- Закройте и откройте EarTrumpet заново
- Перезагрузите Rainmeter

## Дополнительная настройка

### Прослушивание микрофона тоже

Если вы хотите, чтобы визуализатор реагировал на Spotify И на ваш микрофон:

1. В параметрах звука Windows
2. Найдите ваш микрофон
3. Откройте свойства → вкладка **"Прослушивание"**
4. Включите "Прослушивать это устройство"
5. Установите выход на **CABLE Output**

Теперь визуализатор будет слышать и Spotify, и ваш голос!

### Вернуться к стандартной настройке

Чтобы использовать визуализатор со всеми системными звуками:

1. В скине Settings нажмите **"Use default"**
2. Это удалит любой выбор пользовательского устройства
3. Визуализатор будет реагировать на весь системный звук

## Нужна помощь?

Если что-то не работает:
1. Проверьте лог Rainmeter (правый клик Rainmeter → Log)
2. Убедитесь, что названия устройств совпадают точно
3. Попробуйте обновить скин (F5 или правый клик → Обновить)
4. Полностью перезагрузите Rainmeter

Наслаждайтесь вашим Spotify визуализатором! 🎵
